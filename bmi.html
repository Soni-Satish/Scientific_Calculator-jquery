BMI Calculator using jQuery

The Body Mass Index(BMI) calculator can be used to calculate BMI value by using weight and height.By using this calculator anyone can know that he or she is underweight, normal or overweight.We have a table of the readings given by World Health Organisation(WHO).

Table published by WHO

BMI Nutrition Status
Below 18.5 Under weight
18.5-24.9 Normal weight
25.0-29.9 Pre-obesity
30.0-34.9 Obesity class I
35.0-39.9 Obesity class II
Above 40 Obesity class III
